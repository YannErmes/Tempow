⚡ Tempow Help & User Guide

Master task timing, planning, and cloud sync

⏱ Timer Tab — Task Management

Getting Started

The Timer tab is where you manage your daily tasks with precise time allocation.

📸 SCREENSHOT: timer-overview.png - Main timer interface with task list and controls

Creating a Task

  1. Click the + Add Task button in the toolbar
  2. Enter task name (e.g., "Write project brief")
  3. Choose task type:
    • ⏱ Timed - Tasks with a time limit (recommended)
    • ☐ Untimed - Checklist items without time tracking
  4. For timed tasks, set allocated time (max 4 hours)
  5. Optional: Add break time (how long you can pause)
  6. Click Create Task
📸 SCREENSHOT: add-task-modal.png - "New Task" modal showing name, type, and time inputs

Task States & Controls

Each task has 4 states with different controls:

Idle → Task not started
▶ Start button
Running → Timer is counting down
⏸ Pause | ✓ Done (early)
Paused → On break or manually paused
▶ Resume | ⏱ Extend
Done → Task completed
✓ Complete (view only)
📸 SCREENSHOT: task-card-states.png - Show Running, Paused, and Done task cards

Progress Bar & Timer Display

Each task card shows:

  • Progress bar - Visual representation of time spent vs. allocated
  • Timer digits - Time remaining (or time on break if paused)
  • Status pill - Running/Break/Done/Idle indicator
  • Estimated end time - When you'll finish at current pace (only when running)
📸 SCREENSHOT: task-progress-details.png - Close-up of running task with timer, progress bar, and estimated time

Task Actions

All buttons toggle between text and icons: Use ⚙ Settings to choose display mode

  • ⏱ Extend - Add up to 25% more time to a paused task
  • 🔀 Merge - Combine this task with another when priorities change
  • ⇄ Clock - Send task to 24h Clock for scheduling
  • 📝 Notes - Add quick reminders for tomorrow
  • ↓ JSON - Export task as backup
  • ⊘ Discard - Temporarily hide to focus on critical tasks

Filtering & View Modes

Click the ▼ button to expand filters:

  • Filter by state: All, Running, Paused, Done, Idle, Timed, Untimed
  • View modes: List (default), Grid, Card - Pick what works for you!
📸 SCREENSHOT: filter-bar-expanded.png - Expanded filter/view mode panel

Discarded Tasks & Trash

Discard temporarily hides tasks (visible in Discarded section)

Delete moves tasks to Trash (auto-purged after 24 hours)

You can Restore from either at any time

◕ 24h Clock — Timeline Planning

What is the 24h Clock?

A visual timeline of your entire day (24 hours). Plan when you'll work on tasks, when you'll take breaks, and optimize your schedule.

📸 SCREENSHOT: clock-overview.png - Full 24h clock view with example tasks

Adding & Scheduling Tasks

  1. Click + Add Task in the Clock tab
  2. Fill in task details (name, duration, optional break time, color)
  3. On the 24h clock, select a time window where you want to work on it
  4. The system will show you how much time is available in that slot
  5. Click Add Task to schedule it
📸 SCREENSHOT: clock-add-task.png - Add Task modal and clock selection interface

Color Coding

Tasks have different colors for easy visual separation. Click Random to auto-assign, or manually pick colors that match your priorities.

Sending to Timer Tab

Once you've scheduled tasks on the clock:

  1. Start your day (click START MY DAY)
  2. The system automatically creates a task in the Timer tab
  3. Begin working with full time tracking
📸 SCREENSHOT: inbox-preview.png - Timer tab showing incoming tasks from clock with preview

☁ Cloud Sync — Backup & Restore

Why Sync to Cloud?

Backup all your tasks to Google Sheets and access them from any device. Your data is secure and always recoverable.

First-Time Sync Setup

  1. Go to Data section and click ☁ Sync to cloud
  2. Enter your email address (your unique identifier)
  3. Enter a username (alphanumeric, no spaces)
  4. Click ☁ Sync now
  5. Watch the 2D animation while your tasks upload 🌀
  6. You'll get a unique Sync ID - save this!
📸 SCREENSHOT: sync-modal.png - Sync to Cloud modal with email/username fields
📸 SCREENSHOT: sync-animation.png - 2D orbital loading animation during sync

Loading from Cloud

  1. Click ☁ Load from cloud in the Data section
  2. Enter the same email and username you used to sync
  3. Click the button to fetch your tasks
  4. You'll see a preview of how many tasks are ready to load
  5. Click Confirm load to restore them
📸 SCREENSHOT: load-modal-preview.png - Load from Cloud showing preview of tasks to restore

Pro Tips

  • 💾 Sync regularly to keep backups fresh
  • ⚠️ Loading will replace your current tasks, not merge them
  • 🌍 Works offline - syncs when you reconnect
  • 🔐 Your email+username combination is encrypted

📥 Data Management

Exporting Tasks

  • ↓ All tasks JSON - Download your entire task database
  • ↓ Completed only - Export just finished tasks for record-keeping
  • ↓ JSON per task - Right-click > JSON to export single task

Great for backups, sharing, or importing into other tools!

Importing Tasks

  1. Click ↑ Import JSON
  2. Drag & drop your JSON file or click to browse
  3. Preview the tasks to be imported
  4. Click Import to add them
📸 SCREENSHOT: import-modal.png - Import JSON modal with drag-drop zone

Sync Elapsed Time

If the app crashes or closes:

  • Click ↻ Sync elapsed to automatically add lost time to running tasks
  • The system calculates time passed since your last activity

⚙ Display Settings

Button Display Modes

Click the ⚙ Settings button (top right) to toggle:

  • 📋 Text + Icons (default) - Full button labels for clarity
  • 📦 Icons Only - Compact view, shows only icons with tooltips

Your choice saves automatically and applies everywhere!

📸 SCREENSHOT: settings-modal.png - Settings modal showing button mode toggle

✨ Tips & Tricks

Productivity Hacks

  • 📌 Use the Clock tab first - Plan your day, then execute in Timer
  • ⏸ Pause for breathing - Use break time to prevent burnout
  • 🔄 Merge tasks strategically - Combine related work to maintain flow
  • 📝 Add tomorrow's notes - Prep context for the next day
  • ☁ Sync before bed - Always have a backup of today's work
  • 📊 Check Reports weekly - See patterns in your productivity

Common Issues

Q: Timer jumped ahead after closing the app

A: Click ↻ Sync elapsed to account for missed time automatically

Q: I can't find a task I discarded

A: Scroll to the Discarded Tasks section at the bottom of Timer tab

Q: Cloud sync says "Not found"

A: Double-check your email and username match exactly what you used during first sync

Q: How long do tasks stay in Trash?

A: 24 hours. After that, they're permanently deleted. Restore them anytime before that!
